# ⚙️ Cooling Tower Gearbox AI Temperature Prediction System

An AI-powered system that predicts gearbox temperature in cooling towers using Machine Learning and provides real-time monitoring through an interactive dashboard.

---

## 🚀 Overview
This project focuses on predicting gearbox temperature to prevent overheating and failure in cooling tower systems.

It enables early detection of abnormal conditions and supports preventive maintenance.

## 💡 Problem
Cooling tower gearboxes often face overheating issues, leading to:
- Equipment damage  
- Unexpected downtime  
- High maintenance cost  

Traditional monitoring lacks predictive intelligence.

---

## ✅ Solution
This system:
- Predicts gearbox temperature using ML  
- Identifies overheating risk  
- Provides system health insights  
- Displays real-time data in dashboard  

---
